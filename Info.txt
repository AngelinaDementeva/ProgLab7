Папка OrganizationWork - проект, реализованный средствами программы NetBeans 12.4 на языке программирования Java 15.
			Данный проект объединяет в себе клиента и сервера. Для запуска сервера используется класс MainServer,
			а для запуска клиента используется класс MainClien. То есть, в данном проекте объединены и сервер и клиент.


	******Разделенные проекты******

Папка OrganizationWorkServer - проект, реализованный средствами программы NetBeans 12.4 на языке программирования Java 15.
			Данный проект является серверной частью приложения.
Папка OrganizationWorkClient - проект, реализованный средствами программы NetBeans 12.4 на языке программирования Java 15.
			Данный проект является клиентской частью приложения.


PS  Сделано два реализации - в одном проекте клиент и сервер и раздельные проекты для клиента и сервера. Какую реализацию использовать - решайте сами!!!


В папке lib проектов OrganizationWork и OrganizationWorkServer находятся библиотеки для логирования на сервере, которые нужны для выполнения дополнительного задания.
Для хранения коллекции используется база данных PostgreeSQL. Администратор в базе данных всегда имеет ID = 0.
Файл script.txt проектов OrganizationWork и OrganizationWorkClient используется для выполнения команд из файла на клиенте.
